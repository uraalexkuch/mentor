# План реалізації форми замовлення індивідуальної онлайн консультації

## Мета
Реалізувати модальну форму для замовлення індивідуальної онлайн консультації, яка відкривається при натисканні кнопки "Замовити консультацію" на сторінці офісив ДЦЗ.

## Вимоги до форми
Форма повинна включати наступні поля:
1. ПІБ (текстове поле, обов'язкове)
2. Дата народження (поле дати, обов'язкове)
3. Телефон (текстове поле, обов'язкове)
4. Електронна пошта (текстове поле з валідацією email, обов'язкове)
5. Оберіть регіон (випадаючий список з переліком областей)
6. Чи Ваш бізнес зараз діючий? (радіокнопки: Так/Ні)
7. Якщо ні, зазначте причину: (текстова область, видима тільки якщо вибрано "Ні")
8. Інформація щодо консультації: (чекербокси з опціями:
   - Державні програми підтримки бізнесу;
   - Підбір грантових програм на створення або розвиток власної справи;
   - Менторський супровід в отриманні мікрогранту/гранту;
   - Інше (зазначити, що саме) - текстове поле, видиме тільки якщо вибрано "Інше")
9. Бажана дата отримання консультації (випадаючий календар)

## Структура реалізації

### 1. Створення компонента діалогового вікна
Створити новий Angular компонент `dcz-consultation-order-dialog` у директорії:
`D:\web_ptoject\mentor\public-mentorship\src\app\pages\profnavch\dcz-mentorship\dcz-mentorship-consultregion-main\`

Компонент повинен включать:
- HTML шаблон з формою
- TypeScript логіка для обробки форми та валідації
- CSS стилі для оформлення
- Імпорт необхідних Angular Material модулів

### 2. Модифікація головного компонента
Оновити метод `orderConsultation` в `DczMentorshipConsultregionMainComponent` для відкриття діалогового вікна замість простого логування в консоль.

### 3. Реалізація логіки форми
- Валідація всіх обов'язкових полів
- Умовне відображення полів залежно від вибору радіокнопок та чекербоксів
- Відправка форми (в даному випадку - симуляція відправки та показ повідомлення про успіх)
- Повідомлення про успішне відправлення форми

### 4. Інтеграція з существующим кодом
- Додати необхідні імпорти Angular Material модулів
- Оновити розмітку головного компонента для підтримки діалогового вікна

## Технічні деталі

### Компоненти які потрібно створити
1. `dcz-consultation-order-dialog` - головний компонент форми
2. Оновлення існуючих компонентів:
   - `DczMentorshipConsultregionMainComponent` - для відкриття діалогу

### Angular Material модулі потрібні для імпорту
- MatDialog
- MatFormField
- MatInput
- MatDatepicker
- MatNativeDateModule
- MatRadio
- MatSelect
- MatCheckbox
- MatButton
- MatIcon

### Логіка роботи
1. При натисканні на кнопку "Замовити консультацію" відкривается модальное вікно з формою
2. Форма валідується перед відправкою
3. При успішній відправці показується повідомлення про успіх
4. Форма може бути закрита користувачем

## План дفيذ

### Крок 1: Створення компонента діалогового вікна
- Створити папку `dcz-consultation-order-dialog` в директорії компонента
- Створити файли компонента (.ts, .html, .css)
- Реалізувати форму з усіма необхідними полями
- Додати логіку валідації та обробки відправки

### Крок 2: Оновлення головного компонента
- Змінити метод `orderConsultation` для відкриття діалогу замість console.log
- Додати необхідні імпорти (MatDialog, компонент діалогу)

### Крок 3: Тестування
- Перевірити, що форма правильно відображається
- Перевірити валідацію полів
- Перевірити умовне показання полів
- Перевірити відправку формы та показ повідомлення про успіх

## Залежності
Потрібно перевірити, чи вже існують необхідні Angular Material модулі в проекті, та при потребі додати їх у imports компонента.

## Ризики та зменшення ризиків
- Ризик: Конфлікти стилів з існуючими стилями проекту
  - Зменшення: Використовувати scoped стилі для компонента
- Ризик: Проблеми з валідацією форми
  - Зменшення: Тщательно протестувати всі сценarii валідації
- Ризик: Проблеми з умовним показанням полів
  - Зменшення: Ретельно протестувати всі комбінації виборів

## Висновок
Після реалізації цього плану користувачі зможуть замовляти індивідуальну онлайн консультацію через зручну форму, яка задовольняє всі зазначені вимоги.
